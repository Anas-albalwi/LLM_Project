# LLM_Project: Grammar and Spelling Corrector

An AI-powered project designed to automatically correct English grammar and spelling errors using Large Language Models (LLMs).

## Project Overview
This project aims to build a language model capable of understanding common English grammatical patterns and transforming erroneous sentences into their correct forms.

## Dataset Methodology
The project employs a robust data strategy divided into two phases:
1. **Training Dataset:** A collection of (input, target) pairs used to teach the model the underlying grammatical patterns and common language rules.
2. **Testing Dataset:** A set of unseen sentences used to evaluate the model's generalization capability. This ensures that the model has learned the actual grammar rules rather than just memorizing the training data.
